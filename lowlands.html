Lowlands

Frontend development - klik hier voor de werkende site

Situatie

In het tweede jaar van CMD kregen wij een opdracht van lowlands, waarbij wij hun website moesten redesignen. Hiervoor gebruikten wij hmtl, css en Javascript. De uitdaging die hierbij kwam kijken was het gebruik maken van pseudoclasses, om beter te snappen hoe html en css in elkaar zitten. Ook was het belangrijk dat de site responsive werd, zodat elke gebruiker op elk device de site kan bezoeken.

Taak

De opdracht was individueel. Het was mijn taak om de site te programmeren, designen en testen (op user experience). Belangrijk was dat ik geen classes en id's mocht gebruiken, zodat ik meer begrijp van html/css. Deze opdracht was ook de inleiding naar mediaqueries, een css-command dat ervoor zorgt dat je website responsive word.

Activiteiten

Voor de opdracht heb ik veel desk research gedaan naar Lowlands. Ik heb een sitemap gemaakt van de huidige site en gekeken naar de informatie structuur. Ook heb ik veel deskresearch gedaan naar html, css en Javascript. Dankzij deze research ben ik achter veel verschillende dingen gekomen die deze talen voor je kunnen doen.

Resultaat

Het resultaat is een werkende site gemaakt in html, css en Javascript. De code is volledig semantisch correct zonder gebruik te maken van classes en id's. Ook ben ik veel bezig geweest met user experience, vooral bij de nieuwsbrief.
